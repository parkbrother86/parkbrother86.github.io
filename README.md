# parkbrother86.github.io

GitHub Pages에서 `Dinky` 원격 테마를 사용하는 Jekyll 블로그입니다.

## 포함 내용

- `remote_theme: pages-themes/dinky@v0.2.0` 기반 설정
- `assets/css/style.scss`와 로컬 `_layouts/` 오버라이드
- `_includes/head-custom.html`, `_includes/language-switcher.html` 이중언어 토글 보조 코드
- `_posts/` 기술 블로그 포스트
- `about.md`, `topics.md`, `categories/` 주제별 페이지

## 로컬 실행

```powershell
bundle install
bundle exec jekyll serve
```

기본 주소는 `http://127.0.0.1:4000` 입니다.
